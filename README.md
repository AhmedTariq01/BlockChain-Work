## Simple Implementtaion of Blockchain in Python and Node
### Node Implementtaion:
- Run 'npm run' to install all the dependencies.

### Python Implementtaion:
- Install the following libraries:
- pip install datetime
- pip install hashlib
- pip install json
- pip install uuid
- pip install urllib.parse
- pip install flask
